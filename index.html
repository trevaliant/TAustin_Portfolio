Latest Project

Data Conversion

This project took data from one EVV system to another EVV system. There was almost 5 years of data gathered for Medicaid recipients for visits for various home care services. Getting this data into the new system was imperative so providers could have historical data, audits could be performed, and members care could be continuous. The source system did not use the same format, fields or export types that aligned with our system. Here is the a write up with code snipets Github.

  • CSV

Populating DFT Messages

Project to develop a solution to intact DFT messages from flat files and translate to HL7. A need arose for a single client that had a large volume of claims that needed to be translated to HL7 for claims to be sent to clearninghouses. The team I managed came together and found a soultion to use our Intersystems Ensemble interface to ingest flat files and change to HL7, not only for the one client but to be used as a sales addon for future clients.
